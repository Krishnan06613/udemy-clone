# 🎓 Udemy Clone

A responsive Udemy-inspired website clone built using **HTML5** and **CSS3**. This project recreates the basic user interface of the Udemy homepage, including course listings, categories, search bar, recommendations, and footer sections.

## 🚀 Features

- Responsive Navigation Bar
- Search Box UI
- Course Categories Section
- Promotional Banner
- Recommended Courses Section
- Popular Courses Section
- Topics Recommendation Area
- Clean Footer Design
- Font Awesome Icons Integration
- Google Fonts Styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Gabarito)
- Font Awesome Icons


2. Open the project folder:

```bash

```

3. Open `index.html` in your browser.
